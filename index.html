Befter.js – jQuery плагин для сравнения 2х картинок методом наложения одной на другую. Его удобно использовать для отображения различий, например:

  • между версиями (whats new?)
  • сравнения дизайнов
  • различий на старых и новых фото
  • применения фото эффектов

Такой же механизм использовал Илья Бирман для наглядной демонстрации отличий на карте метро

Опции

			$('.comparer').befter({
				color 		: 'red', 	// цвет
				style 		: 'solid',	// стиль линии
				position 	: '50px',	// положение линии-делителя от левого края
				bold 		: '1px', 	// толщина линии
			});

Примеры
