Befter.js – jQuery плагин для сравнения 2-х изображений методом наложения одного на другое. Такой метод удобно использовать для отображения различий, например:

  • между версиями (whats new?)
  • при сравнении дизайнов
  • различий на старых и новых фото
  • при применении фото-эффектов

Такой же механизм использовал Илья Бирман для наглядной демонстрации изменений на карте метро.

Опции

			$('.comparer').befter({
				color 		: 'red', 	// цвет
				style 		: 'solid',	// стиль линии
				position 	: '50px',	// положение линии-делителя от левого края
				bold 		: '1px', 	// толщина линии
			});

Примеры

Простой пример

			$('.comparer').befter({position: '400px'});

От прототипа к дизайну

Пример с кастомизаций

			$('.comparer').befter({color: 'blue', style: 'dashed', bold: '2px', position : '375px'});
